# Usage

```sh
$ npm install -g node-starter
$ node-starter COMMAND
running command...
$ node-starter (-v|--version|version)
node-starter/0.0.0 darwin-x64 node-v10.3.0
$ node-starter --help [COMMAND]
USAGE
  $ node-starter COMMAND
...
```
